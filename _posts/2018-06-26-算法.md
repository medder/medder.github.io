## 1.插入排序

算法思想：扑克牌

[扑克牌](https://github.com/medder/image/blob/master/algorithm/insert.png?raw=true)

算法实现：
```
def insert_sort(l):
    for i in range(1, len(l)):
        key = l[i]
        j = i - 1
        while j >=0 and l[j] > key:
            l[j+1] = l[j]
            j = j - 1
        l[j + 1] = key   // 注意此处需要加1， 因为循环中减了1
    print(l)
```
